Installation

NAnt is shipped in the form of a single .zip file.  Simply unzip the contents into a directory.  To uninstall delete the directory.

If you are installing a new version use a clean directory.  Delete or rename the existing installation directory.

 Important: Modify the system path to include the nant\bin folder.

Once NAnt is installed you can launch it from the command line by typing nant.  To find out what version you are running type: nant -help.
