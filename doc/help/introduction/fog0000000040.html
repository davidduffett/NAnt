Feedback and Contributing

We love to hear from users but please do not abuse individual author's email addresses.  If you would like to ask a question of send feedback please use one of the following methods.

Users

User related questions should be posted to the nant-users list.  It is monitored so if somebody can reply they will.  Having questions posted to this list may allow others having the same problem to solve it on their own. You can subscribe to this list and view the archives by visiting:

 User's Mailing List

Developers

Developer related questions should be posted to the nant-dev list.  You can subscribe to this list and view the archives by visiting:

 Developer's Mailing Lists
