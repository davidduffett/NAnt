Framework Support

NAnt has been tested using the following runtime frameworks :

  • Microsoft .NET Framework 1.0
  • Microsoft .NET Framework 1.1
  • Microsoft .NET Framework 2.0
  • Mono (1.0 and 2.0 profile)
